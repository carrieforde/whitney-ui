## Style API

Use [CSS custom properties](https://developer.mozilla.org/en-US/docs/Web/CSS/Using_CSS_custom_properties) to customize the appearance of `Spinner` to suit your theme. Defaults are shown in the code below.

```css
:root {
  /* Base Spinner styles */
  --spinner-width: var(--base-font-size, 18px);
  --spinner-color: var(--color-default-main, #323f4b);

  /* Sizes */
  --spinner-small-width: var(--base-font-size, 18px);
  --spinner-medium-width: var(--base-font-size, 24px);
  --spinner-large-width: var(--base-font-size, 36px);

  /* Colors */
  --spinner-primary-color: var(--color-primary-main, #35469c);
  --spinner-secondary-color: var(--color-secondary-main, #de3a11);
}
```
